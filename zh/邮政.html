7  邮政服务指南

跟着流程走，就可以寄出一份标准、合格的邮件！

7.1 邮政服务概览

德国邮政（Deutsche Post）是德国境内最主要的邮政服务提供商，隶属于 DHL 集团。其服务涵盖信件、明信片、挂号信、印刷品、包裹等多种邮寄方式。

  • 官网地址：Deutsche Post
  • 建议使用德语界面，英语版功能不完整。

7.4 如何书写标准信封

注意

目前大量国家/地区均已采用机器分拣，请尽量将信封撰写工整，清晰，尽量不要涂抹。

  • 信封：目前国际/国内的平信都建议使用DL信封，即DIN Lang，规格为22*11cm（除非您的邮件并非信纸，或者DL信封对您来说过小。）
  • 寄件人地址：左上角
  • 收件人地址：右下角
    • 如果寄往国内，请在收件人下方填写收件人联系电话
    • 如果不知道对方的邮政编码，可以选择不写，或者选择前往中国邮政邮编查询
  • 邮票粘贴：右上角（可贴多个）
  • 格式要求：
    • 左右下边距 ≥ 15mm，上边距 ≥ 40mm
    • 字体清晰、无高光、无边框
    • 国际邮件：国家名用大写德语/英语/法语书写
  • 可选： 在左下角粘贴您个人的挂号标签/AR标签/航空标签/优先标签

7.5 邮票购买方式

  1. 线下邮局柜台：可直接说明需求（如“internationaler Standardbrief mit Einschreiben”）
  2. Internetmarke 在线邮票：
    • 官网选择产品 → 支付 → 下载 PDF → 打印并贴在信封右上角
    • 所有 Internetmarke 邮票自动包含基础追踪服务（BS）
  3. 数字邮票：
    • 打开手机app
    • 选择下方frankieren
    • 选择合适您的Proto
    • 选择als code zum Beschriften
    • 将app生成的数字写在信封上即可

7.6 寄件流程

  1. 确认您希望寄出什么
  2. 在指定位置写下收件人和寄件人地址
  3. 购买相应资费的邮票，粘贴在合适的地方
    1. 明信片到此处即可丢入邮筒
  4. 如果是没什么重要东西的信件（无重要文书材料，无浓厚情感价值，无运抵期限要求）可选择平信，同时强烈建议使用附加免费的BS（基础函件追踪服务）。
  5. 如果满足以上条件的至少两条，或者经过心理建设依然强烈希望顺利抵达建议走挂号信
  6. 如果是一本书或者一份印刷品，请考虑国际印刷品特殊函件

7.7 常见问题

7.7.1 我能在信封里放什么？

终于下定决心动笔了？非常好！

狭义的信件邮寄建议仅限于纸质文件或卡片。但根据德国邮政的邮寄规定，只要符合以下条件，您也可以邮寄部分轻型小物件：

  • 信封厚度不得超过 5mm，否则可能无法通过自动分拣，被退回或延迟；
  • 挂号信赔偿上限为 €25 欧元，超出价值的物品不建议通过信件邮寄；
  • 金属类物品存在较高风险被识别或拦截，请避免或务必包装遮蔽；
  • 总重量不可超标，务必在平信标准（通常 20g–50g 以内）范围内。

一般来说，小贴纸、票据、打印照片、磁贴等轻薄物件可酌情考虑，但请确保包装平整且无锋利突起。

7.7.2 什么是Basis-Sendungsverfolgung？

这是德国邮政提供的免费基础追踪服务，简称 BS 服务，适用于平信（Standardbrief）和明信片（Postkarte）邮件。

特点如下：

  • 提供 两个追踪节点：
    1. 邮件是否成功被德国邮政接收并进入分拣中心；
    2. 邮件是否被交付至国际交换机构（如发往中国时送入法兰克福国际中心）；
  • 不提供最终签收信息，因此追踪作用相对有限；
  • 免费服务，可通过以下方式启用：
    • 使用印有 矩阵二维码 的实体邮票；
    • 使用 Internetmarke 电子邮票（默认包含此服务）；
    • 线下柜台购买时可向工作人员说明，添加此服务；
  • 通过 Deutsche Post App 扫描邮票二维码后，可自动在 App 中显示追踪进度。
注意

若您贴的是普通图案邮票、或未扫描二维码，该信件可能不具备 BS 服务。

使用 BS 服务，您可以在 App 或官网实时查看邮件是否“出发上路”了。虽然不如挂号信精确，但对平信已是极大的提升。

7.7.3 有哪些附加服务可供选择？

德国邮政提供多种邮件附加服务，您可以根据信件的重要程度与所需保障级别选择合适的服务选项：

  • Basis-Produkt（基础服务） 无附加保障、追踪或签收，仅适合不重要邮件（如贺卡、明信片等）。

  • Prio（优先投递）

    • 含优先运输与增强追踪功能；
    • 需在邮局现场办理，不支持国际邮件；
    • 不含签收保障或赔偿，适合国内时效敏感但非重要邮件。

  • Einschreiben Einwurf（投递式挂号）

    • 投递员将邮件投入收件人信箱，系统记录投递；
    • 限德国境内使用；
    • 含基础追踪与 €20 赔偿；
    • 无需签收。

  • Einschreiben（标准挂号）

    • 投递员需本人或指定代理签收；

    • 含追踪与 €25 赔偿；

    • 支持国际邮件，推荐用于重要文件。

      • Einschreiben Eigenhändig（本人签收）
        • 仅限收件人本人签收，禁止代理；
        • 需额外加 €2.20；
        • 实务中常拒绝 Vollmacht 授权签收；
        • 不适用于国际邮件。
      • Einschreiben Rückschein（签收回执）
        • 邮政寄回签收凭证，含签名与投递时间；
        • 推荐用于法律文书、对外管局沟通等；
        • 价格较高，仅推荐邮局柜台办理；
        • 适用于国际挂号信。

  • Wert（保险挂号）

    • 适用于寄送贵重物品（如 现金 ≤ €100、贵重小件、ebay 交易物品）；
    • 提供更高等级追踪与专门运输；
    • 仅可在邮局线下办理；
    • 附加费用：+ €4.45（不含基础邮资）；
    • 属于高保障服务，适用于需特别保护的邮件。

7.7.4 我的信件到哪了？

  • 电脑端查询： 请访问 德国邮政邮件号追踪平台

  • 手机端查询： 下载 Deutsche Post App，在主页中搜索或扫描邮件上的条形码或矩阵二维码，亦可手动输入挂号号码。

  • 使用限制： 请注意，只有在使用包含追踪功能的附加服务时，这些方法才能查询邮件状态。其他情况无法追踪。

  • 国际传递提示： 如果在官网看到“已送交国际邮件中心即将发往（weiterleiten）中国”等字样，说明中国邮政已成功接收邮件信息。此时，您可以访问 中国邮政官网 的“邮件查询”功能，输入挂号号码以查看邮件在中国境内的运输路径（仅适用于标准挂号邮件）。

7.7.5 我的邮件看起来好像没动静了！咋办？

  • 先别慌！ 如果邮件暂时没有更新，按以下步骤操作：

    1. 等待观察： 根据德国邮政官网的信息，发往亚洲（中国及港澳台）的邮件通常在 10–16 个工作日内送达（非保证时效）。高峰期如遇分拣超负荷，短暂“失联”是正常现象。

    2. 超过 14 个工作日无动静？提交调查： 请访问Nachforschung请求站填写 Nachforschung 表单进行正式查询。

      提交前，请确保保留好所有相关文件，包括邮件照片、收据、付款凭证等。如信件内容特别重要，建议在寄出前扫描备份。

✨ 最后祝愿大家寄出的每一份承载爱意与情感的信件都能顺利抵达心中的那个人。
